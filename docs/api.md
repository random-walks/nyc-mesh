# API Reference

The supported public contract is now subpackage-first:

- `nyc_mesh.models`
- `nyc_mesh.io`
- `nyc_mesh.analysis`
- `nyc_mesh.export`
- `nyc_mesh.pipeline`
- `nyc_mesh.cli`

The root `nyc_mesh` namespace is intentionally minimal and only exposes
`__version__`.
