# Example Template

This folder is the canonical bootstrap template for new `nyc-mesh` examples.

Copy it to a new semantic slug, then replace the placeholders in `main.py`,
`pyproject.toml`, and this README with the story for your new example.

Good real-data mesh examples should document:

- which official dataset URLs they rely on
- which large assets must be staged in `cache/`
- which smaller assets are fetched automatically
- which tracked `reports/` outputs are worth keeping in git
